# Matcha UI

Monorepo for Matcha UI.

## Requirements
- Node >= 20
- pnpm

## Repo layout
- `packages/` — published packages (e.g. `@matcha-ui/components`)
- `apps/` — apps (e.g. docs/Storybook)

## Commands
- `pnpm build` — run builds via Turborepo
- `pnpm test` — run tests via Turborepo
- `pnpm dev` — run dev tasks via Turborepo
